# Movie Review Sentiment Analysis

This project implements a sentiment analysis system using Natural Language Processing (NLP)
techniques to classify movie reviews as positive or negative.

## Technologies Used
- Python
- Google Colab
- NLTK
- scikit-learn
- TF-IDF
- Naive Bayes

## Features
- Text preprocessing
- TF-IDF feature extraction
- Sentiment classification
- Model evaluation
